# 有复验风险的 AI 报告示例

这个文件故意包含错误结论，用来测试 `$comment-analysis-review` 是否能识别弱证据和高风险表达。

## 待复验结论

1. 睡眠呼吸监测占海外社媒讨论总量的 49.9%。

2. Fitbit 口碑最好，因为正面率达到 66.7%。Oura 以 50% 排名第二。

3. TikTok 用户是年轻种草人群，最适合用潮流化产品定位触达。

4. 睡眠佩戴舒适度负面率最高，为 52.3%；夜间生理指标监测负面率第二高，为 69.6%。

5. 既然 CPAP 用户抱怨面罩不舒服，Shokz 可以把产品定位成更舒适的 CPAP 替代方案，并强调医疗级准确性。

## 复验上下文

- 49.9% 实际是 6,230 / 12,489 条产品标注记录，不是全部 35,232 条原始社媒记录。
- Fitbit 只有 3 条评论。
- Oura 只有 10 条评论。
- 夜间生理指标监测有 135 条提及，负面率为 69.6%。
- 睡眠佩戴舒适度有 906 条提及，负面率为 52.3%。
- CPAP 是治疗设备；消费级监测产品不能在没有证据时被定位为治疗替代方案。
